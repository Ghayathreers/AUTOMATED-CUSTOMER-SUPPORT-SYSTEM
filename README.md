# Automated Customer Support System

An AI-powered **Automated Customer Support and Service Resolution** system that intelligently analyzes customer complaints, identifies the relevant customer and order, classifies issues, and provides automated resolution recommendations using **Machine Learning, Deep Learning, Natural Language Processing (NLP), Small Language Models (SLM), Generative AI, and Agentic AI**.

---

## Features

- Intelligent customer complaint analysis
- Smart customer and order lookup
- Automatic issue classification
- Machine Learning & Deep Learning based prediction
- NLP-based sentiment analysis and entity extraction
- SLM-powered complaint summarization
- AI-generated support responses
- Agentic AI decision support and escalation
- Modern enterprise-style dashboard

## Project Workflow

1. Customer submits a support query.
2. AI preprocesses and analyzes the complaint.
3. Customer and order information are matched.
4. Machine Learning and Deep Learning models classify the issue.
5. NLP extracts sentiment and key entities.
6. SLM generates a concise complaint summary.
7. Generative AI creates a personalized resolution.
8. Agentic AI determines whether to resolve automatically or escalate to a support agent.
9. Final resolution is displayed to the user.
